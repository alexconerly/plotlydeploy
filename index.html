This website allows users to select an individual's ID from the dropdown menu to view the volunteer's demographic information and the corresponding bacteria found in that volunteer's navel. The hope of the study is that some of the bacteria could serve as a useful resource for synthesizing proteins to create great-tasting, synthesized meats. Feel free to experiment and compare results from each candidate.
